# CargoApplication
The cargo application provides cargo price tool backend services
## The technologies are used:
- Framework: .net core 3.1
- Database: SQL Server.
- DI: Simple Injector
- Tests: NUnit

# Requirments
 - Net Core 3.1
 - Visual Studio 2019 
 - SQL Server
 
# Usage
## On CommandLine
- Go to /CargoApplication directory. 
- dotnet build
- dotnet run
## On Visual Studio
- Go to /CargoApplication directory. 
- Open CargoApplication.sln
- Click & Start "IIS Express"
